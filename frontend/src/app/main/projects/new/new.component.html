Creating a new project allows you to organize and manage your tasks, team members, and resources effectively. Fill in the required fields to set up your project.

  • Project Name: The name of your project. This is a required field.
  • Project Key: A unique identifier for your project, typically used in URLs and references. Must be uppercase letters and numbers only.
  • Description: A brief overview of the project's purpose and goals.
  • Status: The current state of the project (e.g., Active, Archived, On Hold, Completed).
  • Visibility: Determines who can see the project (e.g., Public, Private, Team (Invite only)).
  • Priority: Indicates the importance of the project relative to others.
  • Start and End Dates: Define the timeline for your project.
  • Color: Choose a color to represent your project visually.
  • Budget: Set a financial limit for the project's expenses.
  • Progress: Indicate the initial completion percentage of the project.

Once you've filled out the form, click "Create Project" to add it to your dashboard. You can always edit these details later as your project evolves.
